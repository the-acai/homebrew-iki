# homebrew-iki

Homebrew tap for [Iki](https://github.com/the-acai/iki) — privacy-first voice dictation for macOS.

## Install

```bash
brew tap the-acai/iki
brew install --cask iki
```

## Update

```bash
brew upgrade --cask iki
```

## Requirements

- Apple Silicon (M1 or later)
- macOS 14 (Sonoma) or newer
